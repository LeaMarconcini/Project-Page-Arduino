Corta Cables

Un proyecto de Arduino

Objetivo

Tenemos como objetivo facilitarle al usuario la repetitiva tarea de medir, cortar y pelar cables de manera manual, transformándola en una tarea automatizada, rápida y precisa, controlada en base a entradas proporcionadas a través de una aplicación móvil bluetooth.

¿A quién beneficia?

Para nosotros este proyecto podría beneficiar a los siguientes individuos:

  • Un profesor, para darle a sus estudiantes cables ya cortados para realizar la práctica de un circuito
  • Un ferretero, para vender cables a medida a sus clientes.
  • Un técnico electrónico o una persona aficionada a la electrónica, para facilitar y agilizar sus proyectos.
